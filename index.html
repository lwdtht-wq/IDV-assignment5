IDV Assignment 5

Overview

When creating the A5 integrated webpage, I first asked GPT to generate a main page based on a “dopamine design” style and made sure the overall visual layout was suitable for placing all four assignments together. Then, I tried to directly merge the full HTML files of A1–A4 into a single page, but because the code was too large and structurally complex (especially A3 and A4, which contain heavy D3 visualizations), GPT couldn’t process all of them at once.

It initially suggested a version with page-to-page navigation, but I wanted A5 to display all content on one continuous page, so I continued looking for alternatives. Directly combining the HTML files would cause CSS, JS, and D3 scope conflicts, so GPT recommended embedding the four assignments separately using <iframe>. This kept all original functionality intact and prevented interference between the files.

Finally, I uploaded index.html along with the four assignment files to GitHub and deployed the site using GitHub Pages. This allowed each iframe to correctly load its corresponding HTML file, and I successfully completed the A5 integrated webpage.
